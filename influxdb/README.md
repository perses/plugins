# InfluxDB Plugin

Plugin for InfluxDB datasource support in Perses.

## Features

- InfluxDB v1.x datasource support
- InfluxDB v3 datasource support
- Time series query support

## Installation

```bash
npm install @perses-dev/influxdb-plugin
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
